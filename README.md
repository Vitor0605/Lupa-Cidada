# Lupa Cidadã 🔍 Aplicação web mobile-first para visualização de dados públicos das cidades do Paraná. ## 🚀 Sobre o Projeto O **Lupa Cidadã** visa democratizar o acesso a indicadores públicos (saúde, educação, segurança, etc.) de forma simples, rápida e transparente. Sem logins, sem cadastros, 100% público. ## 🛠️ Stack Tecnológica - **Front-end**: HTML5, CSS3 (Vanilla), JavaScript (ES6+). - **Back-end**: Node.js 20+, Express.js. - **Cache**: node-cache. - **Dados**: APIs públicas (IBGE, IPARDES, DataSUS, etc.). ## 📦 Estrutura de Pastas ```text frontend/ # Interface do usuário (Mobile-first) backend/ # API REST e integração de dados ``` ## ⚙️ Como executar 1. Instale as dependências: `npm install` 2. Inicie o servidor: `npm run dev` 3. Acesse: `http://localhost:3000` ## 📄 Licença Este projeto é de domínio público para fins de transparência governamental. # Lupa-Cidada # Lupa-Cidada
